# Security Policy

## Supported Versions

Currently, only the latest version of `dbportal` is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | Yes             |
| < 1.0   | No              |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within `dbportal`, please do not open a public issue. Instead, please report it to the maintainers:

1.  **Email**: Send an email to [INSERT EMAIL ADDRESS] with the subject "Security Vulnerability Report".
2.  **Details**: Provide a detailed description of the vulnerability, including steps to reproduce it and any potential impact.
3.  **Confidentiality**: We will acknowledge your report and work with you to resolve the issue privately before making it public.

Thank you for helping keep `dbportal` safe!
